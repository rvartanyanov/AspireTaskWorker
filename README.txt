Welcome to Aspire Task Worker, here are all the instructions needed to open the file:

How to start: 
1.Open Docker Desktop.
2.Run the project from your dev environment.
3.Navigate to the newly opened page and locate the webfrontend project's port.
4.Click the Start Task button to start and add a new task to the list.
5.Copy the Guid and paste it into the textbox then click Check Task to recieve updated information.
6.Add as many tasks concurrently as you preder.

Expected behaviour: 
1.Starting multiple tasks at the same time.
2.Monitoring each task's status by using the Guid and Check Task button.
3.Error handling for inputing a non existing Guid.
4.Real time progress display in %
5.On completion of a task progress must indicate "Completed".
